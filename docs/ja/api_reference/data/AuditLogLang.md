# AuditLogLang(enum)
AuditLogLangは、操作履歴ファイルの言語設定を表します。<br>

### Service
+ [AuditLogService](../services/AuditLogService.md)

| Enumeration | Type | Description | 
|---|---|---|
| JA | xsd:string | 日本語です。 |
| EN | xsd:string | 英語です。 |
