# AuditLogEventType(enum)
AuditLogEventTypeは、操作履歴で取得する操作タイプを表します。<br>

### Service
+ [AuditLogService](../services/AuditLogService.md)

| Enumeration | Type | Description | 
|---|---|---|
| ADD | xsd:string | 追加 |
| SET | xsd:string | 更新 |
| REMOVE | xsd:string | 削除 |
| ALL | xsd:string | すべての操作タイプ |
